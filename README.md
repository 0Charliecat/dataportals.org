## Developer Instructions

This app requires NodeJS (>= v0.8).

1. Clone this repo: https://github.com/okfn/datacatalogs.org
2. Install the dependencies:

        npm install .
3. Try it out locally:

        node app.js

   Then point your browser at http://localhost:5000/
